# Samenvatting eindopdracht 
De eindopdracht is een interactief CSS experiment waarin we iets maken dat we nog niet eerder hebben gemaakt. Het gaat niet alleen om dat het er mooi uitziet, maar vooral dat het laat zien hoe je met CSS iets interactie kunt maken. Het project moet uit een pagina bestaan en interactief zijn zonder JavaScript.

Voor de opdracht zijn er ook een aantal eisen. We moeten minimaal twee nieuwe CSS technieken gebruiken. Daarnaast moet het project een passende titel met goede typografie hebben, werken op verschillende schermgroottes en twee thema’s hebben. Ook is het belangrijk dat we rekening houden met toegankelijkheid, een duidelijke structuur en nette code. 


# Idee beschrijving Koffiemachine bedieningspaneel
Voor de eindopdracht maak ik een interactief bedieningspaneel van een koffiemachine. Met dit paneel kan de gebruiker zelf koffie instellen en bedienen. De gebruiker kan kiezen uit 3 soorten koffie ook zit er een aan en uitknop op om de machine te starten.

Daarnaast kan de gebruiker de temperatuur van de koffie aanpassen zodat de koffie warmer of minder heet wordt. Het doel is dat het lijkt alsof je echt een koffiemachine bedient en dat je meteen ziet wat er verandert als je iets instelt.

# Woensdag reflectie 
Vandaag hebben we een presentatie gehad over color gamut. Ik heb geleerd dat dit alle kleuren zijn die wij als mensen kunnen zien. Om kleuren goed te kunnen defineren in design heb je een  color space nodig. Ook kwam naar voren dat er ongeveer 140 standaard kleurnamen bestaan.

We hebben verschillende manieren besproken om kleuren te definiëren. Zo heb ik geleerd dat de nieuwe manier van RGB schrijven zonder komma’s is, daarnaast hebben we HSL behandeld, saturation verzadiging/grijsheid en lightness, dus hoeveel zwart of wit erin zit.

Verder zijn er meerdere technieken langsgekomen zoals hex codes, gradients en nieuwere methodes zoals OKLCH, color mix en Display P3. Ook hebben we gekeken naar de CSS relative color syntax wat handig is om kleuren flexibel aan te passen.

Na de presentatie ben ik verder gegaan met de layout van mijn koffiemachine. Daarbij heb ik gewerkt aan de interactie, zoals de startknop en het kopje.

# Donderdag reflectie
